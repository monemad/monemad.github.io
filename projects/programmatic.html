Description

Worked in a team to create a coding curriculum based website where users can view courses and lessons supplemented with algorithm visualizations. Used JavaScript, React, and CSS to implement Dijkstra's shortest path algorithm and rendered a visualization using a graph rendered and styled on a grid layout. Authenticated users can edit their profiles, upload profile pictures with AWS S3 integration, and comment/upvote/downvote lessons.

Technologies Used


React | Redux | Node.js | Python | Flask | SQLAlchemy | PostgreSQL | HTML / CSS | AWS

Project Highlights

  • Instantiated a graph using JavaScript to implement Dijktra's algorithm to find the shortest path between two nodes. Constructed a grid using React and CSS to visually render the path of the algorithm.
  • Created an intuitive upvote/downvote system using JavaScript DOM manipulation, Redux state, and CSS to provide users visual feedback of their votes and allowed for editing or removing a vote by dispatching fetch requests to the API server.
  • Designed an interactive and easily navigable profile page where users can view their learning objectives, toggle their completion status, and remove them entirely by implementing Redux state integration and fetching to the API database.
