Tune Lagoon



See it on GitHub Try it on Heroku

Description

A music streaming website inspired by SoundCloud where users can upload songs, create playlists, and comment on other people's uploads. Implemented song upload using AWS S3 and utilized the npm package React H5 Audio Player to aid in uninterrupted music playback as users navigate throughout the site.

Technologies Used


React | Redux | Express | Node.js | Sequelize | PostgreSQL | HTML / CSS | AWS

Project Highlights

  • Implemented the React H5 Audio Player npm package and structured the React component rendering to prevent the player from being subject to re-render. This gives the user uninterrupted playback as they navigate the website.
  • Utilized React Context to access and control the currently playing song, allowing for song playback from any component. Further, comments on the currently playing song have clickable timestamps based on when the user commented.
  • Developed a dynamic drop-down search bar where the user can search for songs, users, and playlists by typing in search query. The results are displayed in real-time with the matching text highlighted for user convenience.
